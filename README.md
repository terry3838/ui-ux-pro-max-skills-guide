# UI/UX Pro Max Skills Guide

## 카테고리 소개
이 가이드는 `ui-ux-pro-max` 에이전트 스킬을 활용하여 개발자, PM, 디자이너가 최고 품질의 UI/UX 디자인을 구현할 수 있도록 돕는 실전 가이드입니다. 50개 이상의 스타일, 161개의 컬러 팔레트, 57개의 폰트 조합 등 방대한 데이터를 바탕으로 체계적인 디자인 의사결정을 지원합니다.

## 대상 독자
- **개발자/PM**: 디자인 감각이 부족하더라도 프롬프트만으로 전문가 수준의 UI/UX를 구현하고자 하는 분
- **디자이너**: 다양한 스타일과 컴포넌트 조합을 AI를 통해 빠르게 프로토타이핑하고 아이디어를 얻고 싶은 분
- **일반 사용자**: 자신만의 웹/앱 아이디어를 시각적으로 완성도 있게 구현하고자 하는 비개발자

## 카테고리 구성
1. **[UI 스타일 (UI Styles)](categories/ui-styles.md)**: 미니멀리즘, 뉴모피즘 등 50+ 시각적 스타일 가이드
2. **[컬러와 폰트 (Colors & Fonts)](categories/colors-fonts.md)**: 161개 팔레트와 57개 폰트 페어링
3. **[UX 가이드라인 (UX Guidelines)](categories/ux-guidelines.md)**: 사용성을 위한 핵심 Do/Don't 체크리스트
4. **[컴포넌트 및 아이콘 (Components)](categories/components.md)**: 차트, 아이콘, 앱 인터페이스 구성 요소
5. **[제품 및 랜딩 페이지 (Products & Landing)](categories/products-landing.md)**: 제품군별 구조 및 랜딩 페이지 최적화
